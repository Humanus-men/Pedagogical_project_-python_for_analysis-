# Проект: Анализ резюме из HeadHunter

## Проблема

  При заполнении анкет, частая проблема резюме - это незаполненная графа с желаемой заработной платой соискателя. 
Компания HeadHunter хочет построить модель, которая бы автоматически определяла примерный уровень заработной платы, подходящей пользователю, исходя из информации, которую он указал о себе.

## Задача

  Подготовка данных для последующего построения модели автоматического определения зароботной платы соискателям, исходя из информации, которую они указали о себе.

## Решение задачи

  1) базовый анализ структуры данных

  2) преобразование данных

  3) разведывательный анализ

  4) очистка данных
